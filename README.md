# Paper-translation 论文翻译小助手
	
Automatic translation of Chinese and English papers
自动翻译中英文论文
复制即可翻译

## requirement
Based on PyQt5, python3

install by pip just like:
```bash
pip install clipboard pyqt5
```

## how to run
run:
```bash
python paperTran.py
```

## features
- detect clipboard
- auto replace ‘\n’
